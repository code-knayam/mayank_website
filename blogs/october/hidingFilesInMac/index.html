Hiding Files In MAC

There's always some stuff that should be hidden !!

8th October, 2017

Everyone has some personal files that we need to hide containing personal / important info or some STUFF ;)
This STUFF is usually kept inside a complex folder strucutre. Hidden folder inside folder in a recursive manner.

You can do this just by a single command on the terminal -

  • Open the terminal - Press command + space

  • Type in terminal and press return. Terminal window would open.

  • Now type in the following lines.

    defaults write com.apple.finder AppleShowAllFiles YES

    Press return
    Restart the Finder, by pressing Option key and right click on Finder icon.
    Then press Relaunch.

    Now all the hidden files would be visible if you go inside the Finder.

  • In MAC, files or folders beginning with a dot ( . ) are considered to be hidden. So just rename your file, which you want to hide, to begin with a dot ( . )
    Ex-> .hidden

    Now, to hide the files, again open the terminal as above. And this time type in this line of code ->
    defaults write com.apple.finder AppleShowAllFiles NO

    Press return

    This would hide all the files, beginning with a dot ( . )

Great job. You just secured all your STUFF.
